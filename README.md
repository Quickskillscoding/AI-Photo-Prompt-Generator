This tool helps digital creators, designers, and social media managers generate optimized AI art prompts based on a database of curated visual components: art styles, subjects, moods, lightings, and extra modifier tags.

## 🚀 Features

- **288 Hand-Crafted Base Prompts**: High-quality combinations across 12 distinct art styles and 8 subjects (3 unique prompts per combination).
- **Platform Customization**: Adjusts prompt recommendations and output formats tailored for Instagram, TikTok, Pinterest, YouTube, and more, complete with aspect-ratio recommendations.
- **No-Repeat Rotation**: Smart regeneration logic prevents consecutive duplicates when generating prompts with the same parameters.
- **Interactive Modifiers**: Click to toggle optional detail modifiers (e.g., *ultra detailed*, *bokeh background*, *8K resolution*).
- **Recent History Log**: Keep track of your last 10 generated prompts with per-history instant copy buttons and full-text previews.
- **Statistics Counter**: Tracks total prompts generated during your session (persisted via browser `localStorage`).
- **Premium Dark Aesthetic**: Styled with a dark glassmorphic design, vibrant neon accent colors (`#A5F302`), clean typography (Inter & Syne), and smooth transition animations.

## 🛠️ How to Run Locally

Since the project uses vanilla HTML, CSS, and JavaScript, you can run it without any build tools or dependencies:

1. Clone this repository or download the files.
2. Open `index.html` in any modern web browser.
3. (Optional) Run it via a local web server (e.g., Live Server in VS Code, XAMPP, or python `http.server`):
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000`.
